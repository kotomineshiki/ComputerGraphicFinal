## 简介
我们组的大作业实现了一个基于openGL的海底场景，包括珊瑚、鱼群、火山、气泡、光晕等等要素。

## 操作方式
WASD控制前后左右移动。鼠标滚轮可以进行镜头缩放。

## 实现功能

我们实现的功能包括

### Basic

- Camera Roaming
- Simple lighting and shading
- Texture mapping
- Shadow mapping
- Model import & Mesh viewing 

### Bonus
- Sky Box
- Text Display
- Gamma correction
- Skeletal animation
- Particle System
- Fluid Simulation
- Collision detection
- Anti-Aliasing
- Lens Flare
